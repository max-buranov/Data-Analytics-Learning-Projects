На основании открытых данных я исследовал то, как выглядит рынок общественного питания в Москве. Я выявил, какие типы заведений преобладают в городе, их количество посадочных мест, соотношение сетевых и несетевых заведений. По результатам исследования была подготовлена презентация. 

В работе и построении графиков мною были использованы библиотеки Plotly, Seaborn, Requests и API Яндекс.Геокодера.
